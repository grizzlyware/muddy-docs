# Muddy Booking Colour Scheme Settings

## Location
- URL: `/settings/branding`
- Navigation: Settings > Branding > Colour palette section

## Available Colour Schemes
1. Muddy (default)
2. Frosty Paws
3. Pawsh Pink  
4. Red Set Go!
5. Pumpkin Pup
6. Bark Mode
7. Noble Paws

## How it Works
- Colour palette applies to both booking forms AND admin area
- Simple radio button selection interface
- Click Save to apply changes
- Changes take effect immediately after saving
- Returns to main Settings page after successful save

## User Experience
- No preview shown - users select by name only
- No visual feedback during selection (just radio button selection)
- Changes are applied system-wide once saved
