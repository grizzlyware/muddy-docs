# Muddy Booking Loyalty Programme Structure

## URL and Location
- Main loyalty programme settings: `/manage/operators/{id}/settings/loyalty`
- Accessed via Settings > Loyalty programme (in Pricing section)

## Setup Process
1. Initial state shows "Set up loyalty programme" button
2. After setup, shows toggle for "Enable loyalty programme"
3. Configuration options available before enabling

## Configuration Options

### Earning Style
- Fixed points per booking
- Points per amount spent

### Basic Settings
- Points per booking (default appears to be 1)
- Reward threshold (default appears to be 10 points)

### Reward Configuration
- Reward type: Voucher or Discount
- Reward value: Fixed monetary value (£5.00 default)
- Reward expiry: Optional setting

### Trigger Mode
- Automatic: Points awarded automatically once booking has started
- QR check-in: Alternative trigger method

### Additional Features
- Earning rules: Control which bookings earn loyalty points
- Stamp card icon: Custom icon for customer display
- Option to add specific earning rules

## Enable/Disable
- Toggle to enable/disable programme
- Disabling "freezes all point accrual. Existing points are preserved"
- Save changes button required to apply settings
