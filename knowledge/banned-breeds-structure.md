# Muddy Booking Banned Breeds Structure

## URL and Location
- Banned breeds settings: `/manage/operators/{id}/settings/banned-breeds`
- Access: Settings > Pricing section > "Banned breeds" option

## Settings Options
### Banned breeds policy (Radio buttons)
- **Allow all** - Permits all dog breeds
- **Allow none** - Restricts all banned breeds  
- **Allow some** - Shows breed selection checkboxes
- **Unspecified** - No policy set

### Allowed breeds (appears when "Allow some" selected)
Checkbox list of banned breeds that can be specifically allowed:
- Pit Bull Terrier
- Japanese Tosa
- Dogo Argentino
- Fila Brasileiro
- XL Bully

### Additional information
- Free text area for custom policy explanation
- Displayed to customers during booking process
- Optional field for providing context or contact info

## Interface Elements
- Standard form layout with radio button groups
- Conditional display of breed checkboxes
- Text area for additional information
- Save button to apply changes
